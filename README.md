# ReactMusic
> a music player hacked by react + redux + material-ui + vw adjust + eslint
# run
```bash
  $ git clone https://github.com/8bulbs/react-music.git
  $ cd react-music
  $ cd api-server
  $ npm i
  $ node app
  $ cd ..
  $ npm i
  $ npm start
 ```
